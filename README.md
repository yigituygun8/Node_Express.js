
# Node.js, Express & React Notes Repository

>This repository contains notes, code samples, and resources for learning and practicing Node.js, Express, React, HTTP methods, CRUD operations, REST API design, and related backend development concepts. It serves as a personal reference and learning journal for my journey in backend and frontend development.


## 📚 Courses

### CTIS 256 - Backend Development
**Topics Covered:**
- Backend & HTTP Fundamentals
- Node.js Basics: npm, modules, HTTP server
- Express Framework: basics, routing, middleware
- View Engine (EJS): GET/POST processing
- Error Handling & Server-side Validation
- Sticky Web Forms
- Application Use Cases & Request–Response Flow
- MySQL Basics: Docker containers, environment variables, connection pooling, simple queries
- CRUD Operations: async/await
- File Upload & Multipart/form-data (Multer)
- Cookies and Sessions
- Authentication: session-based, hashing (bcrypt), JWT (conceptual)
- Fetch API & CORS
- REST API Design & GraphQL (conceptual)
- Single Page Application (SPA) Concepts
- Web Security Fundamentals: XSS, SQLi, CSRF

### React Course
**Topics Covered:**
- React fundamentals: components, props, state
- JSX syntax
- Event handling
- Conditional rendering
- Lists and keys
- Forms and controlled components
- Lifecycle methods & hooks (useState, useEffect, etc.)
- Context API
- Routing with React Router
- Fetching data from APIs
- State management (Redux basics)
- Building and deploying React apps

## 🚀 Getting Started
1. Clone this repository.
2. Install Node.js (LTS recommended) and optionally Node.js/npm for React projects.
3. Use `npm install` in project folders with a `package.json`.
4. Explore the notes and code samples by topic (Node.js, Express, React).


## 📦 Dependencies
Dependencies will be listed in each project folder's `package.json` as needed (e.g., Express, EJS, MySQL, Multer, bcrypt, React, Redux, React Router).
